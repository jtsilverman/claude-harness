# Voice corpus

Empty template. The sharpen loop populates this file with samples of the operator's
voice and phrasing, lifted from session transcripts by `scripts/user_corpus_extract.py`,
so the COO can match tone in user-facing output.

It ships empty: on first runs the corpus is built from your own sessions. Each entry is
a short verbatim quote plus a one-line note on what it illustrates (cadence, vocabulary,
a recurring correction). Do not hand-author entries; let the loop accrete them.
